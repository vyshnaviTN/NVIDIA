## 📖 Overview

This repository documents my learning journey through the **Artificial Intelligence & High-Performance Computing Summer Internship Program** conducted by the **Presidency School of AI and Advanced Computing** in collaboration with the **NVIDIA AI Centre of Excellence**.

The internship combines theoretical foundations with practical GPU-accelerated computing, covering Machine Learning, Deep Learning, Large Language Models, Generative AI, CUDA Programming, GPU Optimization, and High-Performance Computing using the NVIDIA H200 Tensor Core GPU infrastructure.

This repository contains my daily notes, lab exercises, assignments, projects, and learning resources completed throughout the internship.

---

## 🎯 Program Highlights

- 🗓 Duration: **2 Weeks (14 Days)**
- 🎓 Academic Credits: **2 Credits**
- ⏱ Contact Hours: **30 Hours**
- 💻 Format: Lecture + Hands-on GPU Labs
- ⚡ Hardware: NVIDIA H200 Tensor Core GPU
- 🏫 Institution: Presidency University – AI Centre of Excellence
